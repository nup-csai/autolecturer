# Проект "Говорящий Аватар"

Этот проект предоставляет инструменты для создания говорящего аватара на основе статичного изображения и аудиофайла, а также для извлечения текста из видео.

## Установка зависимостей

Для работы программы вам понадобятся следующие библиотеки:

```bash
pip install opencv-python numpy tqdm gtts SpeechRecognition
```

Также требуется ffmpeg для обработки аудио и видео. Установите его согласно вашей операционной системе.

## Структура проекта

```
avatar_project/
│
├── Wav2Lip/              # Директория с адаптированными файлами Wav2Lip
│   ├── inputs/           # Входные файлы для Wav2Lip
│   │   ├── audios/       # Директория для видеофайлов для извлечения текста
│   │   ├── img.png       # Изображение лица
│   │   ├── speech.mp3    # Аудиофайл MP3
│   │   └── speech.wav    # Аудиофайл WAV
│   │
│   ├── outputs/          # Выходное видео
│   │   └── result.mp4    # Результат работы Wav2Lip
│   │
│   └── temp/             # Временные файлы
│
├── src/                  # Исходный код проекта
│   ├── audio_transcriber.py  # Класс для извлечения и транскрибации аудио
│   └── video_processor.py    # Класс для обработки видео и извлечения текста
│
├── modified_inference.py # Модифицированный скрипт Wav2Lip для создания видео
├── run_avatar.py         # Основной скрипт для запуска проекта
└── sample_text.txt       # Файл с текстом для создания аватара
```

## Функциональность

- Извлечение текста из видеофайлов с помощью распознавания речи
- Создание говорящего аватара, который озвучивает текст
- Преобразование текста в речь с помощью gTTS
- Синхронизация движения губ с речью с использованием Wav2Lip

## Использование

### Извлечение текста из видео

1. Поместите видеофайл в директорию `Wav2Lip/inputs/audios/`
2. Запустите скрипт с флагом --extract:
```
python run_avatar.py --extract
```
3. Извлеченный текст будет сохранен в файл `sample_text.txt`

### Создание говорящего аватара

1. Отредактируйте файл `sample_text.txt` и добавьте текст, который должен озвучить аватар
2. Убедитесь, что изображение лица находится в `Wav2Lip/inputs/img.png`
3. Запустите скрипт с флагом --avatar:
```
python run_avatar.py --avatar
```
4. Результат будет сохранен в файл `Wav2Lip/outputs/result.mp4`

### Запуск обоих режимов

Для запуска обоих режимов (извлечение текста и создание аватара) просто запустите скрипт без аргументов:
```
python run_avatar.py
```

## Примечания

- Упрощенная версия создает анимацию без использования нейронных сетей
- Для работы распознавания речи требуется соединение с интернетом
- Качество распознавания зависит от чистоты речи в видео
- Поддерживаются различные форматы видео (.mp4, .avi, .mov, .mkv, .webm)
- Изображение лица должно быть хорошего качества, с четко видимыми чертами лица
- Для лучших результатов используйте четкую аудиозапись с минимальным фоновым шумом
